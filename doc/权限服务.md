## 权限服务

### API

| 方法   | URL             | 可能返回的状态码    |
| ------ | --------------- | ------------------- |
| POST   | /token          | 200,400,401,403,500 |
| DELETE | /token/:user-id | 200,400,401,403,500 |
|        |                 |                     |

权限服务提供用户登陆和用户注销的接口，使用的数据库对象还是user模块的数据库对象
